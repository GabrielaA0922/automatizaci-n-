Implementación de Funciones Matemáticas en C++

Proyecto realizado por Gabriela Campo Ayala, enfocado en la creación de funciones matemáticas básicas en C++ sin usar librerías como <cmath>. Incluye operaciones como valor absoluto, raíz cuadrada (método babilónico), exponencial, normalización y funciones trigonométricas (seno y coseno) mediante aproximaciones.

El código está organizado en la carpeta src, separando declaraciones, implementación y ejecución, y utiliza un Makefile para facilitar la compilación.

Para ejecutarlo en Linux o Windows (MinGW/WSL):

make build para compilar
make run para ejecutar
make clean para limpiar archivos

Este proyecto refuerza conceptos básicos de programación y métodos matemáticos en C++.
